# School ERP

A School ERP system built using Firebase to manage:

- Students
- Teachers
- Fees
- Attendance

## Features
- Student management
- Teacher records
- Fee tracking
- Attendance system
- Firebase Firestore backend

## Tech Stack
- Firebase (Firestore)
- Web / Android frontend (to be added)

## Goal
This project aims to create a simple School ERP system that can be used by small schools to manage daily operations digitally.
